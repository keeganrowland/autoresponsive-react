## Examples

Live demo & docs: [xudafeng.github.io/autoresponsive-react](https://xudafeng.github.io/autoresponsive-react/)

To build the examples local, please run:

```shell
$ make install
$ make server
```

## Installation

```shell
$ npm install autoresponsive-react --save
```
